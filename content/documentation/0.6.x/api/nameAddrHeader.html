Class @JsSIP.NameAddrHeader@ An instance of the @JsSIP.NameAddrHeader@ class contains a "name-addr" or a "addr-spec" value as stated in the BNF grammar of the "RFC 3261":http://tools.ietf.org/html/rfc3261. A "name-addr" value is in the form "display name <URI>". Example: CODE_BEGIN "no-highlight" From: "Alice" CODE_END A "addr-spec" value is in the form "URI".
